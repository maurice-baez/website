Jobly

Fullstack JavaScript job searching app built using React, Node/Express, and PostgreSQL. Features include login/signup, browse and apply for jobs, search jobs by company name using live search, and view the jobs posted by each company.

Demo | Github
Warbler

Warbler is a Twitter clone built with Python, Flask, and PostgreSQL. Users can sign up/login, follow users, be followed by users, and create tweets called “warbles”. Other features added include; liking and unliking posts and functionality to edit the users profile and change the existing password.

Daily Workout App

A web scraping application that sends users a text message every morning with the CrossFit.com's daily workout. Built using Python, Beautiful Soup, and the Twilio api.
